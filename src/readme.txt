Enables Kit embeds to be used in your WordPress blog.

== Description ==

This plugin adds oEmbed support for pasting a link to a kit page on kit.com. The
kits URL must be by itself on its own line for this to work.

See the Kit plugin documentation for more:
https://github.com/kitinc/kit-wordpress-plugin

== Automatic Installation ==

1. In Admin Dashboard, go to Plugins > Add New.
2. Search for "Kit WordPress Plugin".
3. Find the plugin and click "Install".
